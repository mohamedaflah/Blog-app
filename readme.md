# Instruction to setup this project

1. git clone https://github.com/mohamedaflah/Blog-app.git
2. cd ui
   -> npm install
   -> setup .env based on .env-sample
   -> npm run dev
3. open new Terminal without closing existing terminal
4. cd server
-> npm install
-> setup .env based .env.local
-> npm run dev
